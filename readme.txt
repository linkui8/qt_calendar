Для работы проекта необходима установка модуля PyQt6 и PyQt6-tools
	python -m pip install pyqt6
	python -m pip install pyqt6-tools

Проект представляет календарь, в котором есть возможность записи событий по числам, и вывод ближайших событий на экран
Используется База Данных sqlite3 и Дизайнер pyqtDesigner
